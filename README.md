# Sokoban

It is a fanmade game of the famous game [Sokoban](https://ru.wikipedia.org/wiki/Sokoban).

## Usage

- Open solution 
- Select Configuration Mode: Release
- Target Platform: x86

## Controls

- Moving: 
	- W - up;
	- A - left;
	- S - down;
	- D - right
- Restart: R
